Berkeley Brew API

Welcome to the Berkeley Brew API. This is the backend service for the Berkeley Brew application.

Available Endpoints

GET /api/health

Check the health status of the API.

GET /api/cafes

Get a list of all cafes.

GET /api/cafes/:id

Get details for a specific cafe by ID.

GET /api/cafes/:id/reviews

Get reviews for a specific cafe by ID.

For more information, please refer to the GitHub repository.
